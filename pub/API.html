User Functions Inputs Description
makeNevBar / This function creates a div with id of 'nevBar', which is the basic container for all the categories.
makeSideBar / This function creates a div with id of 'sideBar', which is the floating box that's going to hold all the categories in the Navigation Bar.
makeSideStepBar String title This function creates a box that floats at the left of the screen. It's the container that holds all the text information, so that it works as a memo for the user.
addNevBarContent String name, String href This function takes the name of a category and the hyper-referece link as input. It adds an 'a' element in the Navigation Bar created by 'makeNevBar'. That 'a' links to the destination pointed by input 'href'.
addSideBarContent String name, String href This function takes the name of a category and the hyper-referece link as input. It adds an 'a' element in the floating side bar created by 'makeSideBar'. That 'a' links to the destination pointed by input 'href'.
addSideStepBarContent String name This function takes String 'name' as input and add it to the floatng box created by makeSideStepBar.
